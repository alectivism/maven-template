## Search Strategy
1. **Identify the right site** based on the document topic
2. **Search within the site** using SharePoint search
3. **Browse document libraries** if search doesn't find it
4. **Check Shared Documents** — most sites use this as the primary library

## Common Document Locations
- **Brand Kit:** MMAGlobal > Shared Documents > General > Marketing + Media Alliance MMA Brand Kit 2025
- **Research reports:** Research > Shared Documents
- **Event materials:** Events > Shared Documents > [Event Name]
- **Financial docs:** Finance > Shared Documents
- **Board materials:** Board > Shared Documents

## Process
1. Ask what document or information is needed
2. Identify the most likely SharePoint site
3. Search or browse for the document
4. Return the document link or content
5. If not found, suggest alternative locations or who might know
