# Email Drafting — MMA Style

## Formatting Standards
- Font: Aptos 12pt (Outlook default) — do not specify other fonts
- Use "Hi all," for group emails unless addressing specific individuals
- For 1:1 emails, use "Hi [First Name],"
- Sign-off: Match the sender's typical style. Default: "Best," or "Thanks,"
- Keep paragraphs short (2-3 sentences max)
- Use bullet points for action items or lists

## Voice & Tone
MMA's voice is the Hero archetype: energetic, sharp, smart, clear, confident, generous.

- Lead with the purpose/ask in the first sentence
- Be direct — respect the reader's time
- Use active voice
- Include specific dates, numbers, and next steps
- No filler phrases ("I hope this email finds you well", "Just wanted to touch base")

## Audience-Specific Guidelines

### Internal (MMA Staff)
- Casual but professional
- Reference Slack channels, Asana tasks, SharePoint links as appropriate
- Use first names

### Members (CMOs, Marketing Leaders)
- Professional, peer-level tone
- Reference MMA programs by full name on first mention, then acronym
- Highlight value and outcomes, not process
- Include relevant data points (CAP +160% lift, MMGF +50% ROAS, etc.)

### Board Members
- Executive-level brevity
- Lead with impact/decision needed
- Bullet key points
- Keep under 200 words when possible

### External / Prospects
- Include MMA boilerplate in signature area
- Reference member count (825+), reach (350,000+ marketers)
- Focus on peer-driven collaboration and evidence-based approaches

## Process
1. Ask who the audience is if not clear
2. Ask for key points to include
3. Draft the email with proper formatting
4. Present for review before any sending action
5. **Always confirm before sending** — state recipient, subject, and key content

## Output Format
Present the draft as:
- **To:** [recipients]
- **Subject:** [subject line]
- **Body:** [formatted email]
