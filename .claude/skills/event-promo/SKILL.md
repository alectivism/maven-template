## Promotion Formats

### Email Invitation
- Subject line: benefit-focused, include event name
- Body: 3-paragraph max (what, why you, CTA)
- Include: date, location, registration link
- For CCS/CATS: emphasize exclusivity and peer-only environment

### Social Post
- Lead with the value proposition, not the event name
- Include key stats (attendee count, NPS, partner count)
- Use event-specific hashtags

### One-Pager / Sell Sheet
- Header: event name + tagline
- Key stats in callout boxes
- Agenda highlights (3-5 bullets)
- Testimonial quote from past attendee
- Registration CTA with deadline

### Speaker/Sponsor Outreach
- Lead with audience quality and reach
- Include demographics (CMO-level, Fortune 500 representation)
- Reference past sponsors/speakers by name if public

## Process
1. Identify which event and what promotional format
2. Confirm target audience (members, prospects, sponsors, speakers)
3. Draft content with event-specific messaging
4. Include all logistics (date, location, link)
5. Present for review
