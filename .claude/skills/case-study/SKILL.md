# Case Study Generator — MMA

## Case Study Structure

### Standard MMA Case Study Format

**Title:** [Brand/Company] + [Key Result] + [Program]
*Example: "Kroger Achieves 259% Lift with AI Personalization Through MMA's CAP Lab"*

#### The Challenge
- 2-3 sentences on what the brand was trying to solve
- Industry context — why is this a common challenge?
- What had they tried before?

#### The Approach
- Which MMA program or lab was used
- Methodology (brief, accessible — avoid jargon)
- Timeline and scope
- Partner involvement (Claritas, Monks, Vidmob, etc.)

#### The Results
- **Lead with the headline number** — bold and prominent
- Supporting metrics in bullets
- Comparison to industry benchmarks or BAU
- Statistical confidence if available

#### Key Takeaways
- 2-3 actionable insights other marketers can apply
- What made this work (conditions for success)
- What surprised the team

#### About [Brand]
- 1-2 sentence company description

#### About MMA's [Program]
- 1-2 sentence program description
- How to participate or learn more
- Contact information

### Quick-Hit Format (for newsletters, social, sales)
**Length:** 150-200 words
**Structure:**
- Brand + challenge (1 sentence)
- MMA solution (1 sentence)
- Result (1 sentence with headline number)
- CTA (1 sentence)

## Data Sources for Case Studies
- Lab experiment results (CAP, A3, ACE, SIFT, AURA)
- Member testimonials from events (CMO Summit, POSSIBLE)
- Think tank program outcomes (MMGF, BaP, MOSTT assessments)
- DAM/MAC certification outcomes

## Guidelines
- **Get approval** before using a brand's name in a case study
- Verify all numbers against source data
- Don't fabricate quotes — use real ones or omit
- Frame results honestly — include context (sample size, conditions)
- Make it useful, not just promotional — readers should learn something

## Process
1. Identify the subject (which brand, program, result)
2. Gather data from internal sources (SharePoint, lab reports, meeting notes)
3. Draft in the appropriate format
4. Flag any numbers that need verification
5. Note if brand approval is needed before publication
6. Present for review
