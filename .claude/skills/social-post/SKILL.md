# Social Media Content — MMA Standards

## Platform Priority
LinkedIn is MMA's primary social platform. All posts should be optimized for LinkedIn unless otherwise specified.

## LinkedIn Post Structure

### Standard Post (engagement-focused)
```
[Hook line — provocative stat, question, or bold claim]

[2-3 sentences of context/insight]

[Key data point or finding from MMA research]

[Call to action — link, event, or question]

#MarketingInnovation #MMA #[TopicHashtag]
```

### Event Promotion
```
[Event name + value proposition in one line]

[What attendees will get — be specific]
[Notable speakers or content highlights]

[Date, location, registration link]

#POSSIBLE2026 #MMA #MarketingLeadership
```

### Research/Lab Results
```
[Headline finding — lead with the number]

[Context: what was tested, with whom]

[Why it matters for marketers]

[Link to full report or program page]

#MarketingScience #AI #MMA
```

## Voice Guidelines
- **Sharp, not corporate** — write like a smart colleague sharing insights, not a press release
- **Lead with the surprising number** — "+160% lift" beats "Our new research shows..."
- **Ask questions** — drive comments and engagement
- **Tag people and companies** when referencing their work (ask for approval first)
- **Use line breaks** liberally — LinkedIn rewards scannable formatting

## Hashtag Strategy
Primary: #MMA #MarketingInnovation
Topic-specific: #AIMarketing #MarketingMeasurement #CreativeAI #MarketingLeadership #CMO
Event: #POSSIBLE2026 #SMARTIES #CMOSummit
Program: #MovableMiddles #BrandAsPerformance

## Content Calendar Hooks
- Lab results → research posts
- Event announcements → promo posts
- Board member quotes → thought leadership
- DAM/MAC completions → community posts
- Podcast episodes → teaser posts

## Process
1. Clarify platform, topic, and goal (awareness, engagement, registration)
2. Draft 2-3 post variations
3. Include hashtag recommendations
4. Present for review — **always confirm before posting**
